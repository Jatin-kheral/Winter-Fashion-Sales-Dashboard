I design insights that speak louder than numbers.

This project presents a visually engaging and interactive dashboard focused on winter fashion sales trends. It highlights customer preferences, popular winter categories, color and fabric choices, and brand performance across the season. The dashboard offers a simple yet powerful overview of how winter demand shifts month by month.
